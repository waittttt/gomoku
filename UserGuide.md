# Gomoku – User Guide

This document provides instructions on how to play the Gomoku game.

---

## Launching the Game

1. Launch the provided executable file.
2. The character selection screen will be displayed.


---

## Character Selection

1. On the start screen, choose your character by clicking on the character icon.
2. Click the **Start** button to enter the game.

---

## Gameplay

- The game is played on a 15×15 board.
- Players take turns placing chess pieces on empty grid cells.
- The first player to connect five consecutive chess pieces horizontally, vertically, or diagonally wins the round.
- During gameplay, the current score is displayed on the screen.

---

## Controls

| Action | Input |
|------|------|
| Select character | Left mouse click |
| Place chess piece | Left mouse click on board grid |
| Restart current round | Click **Restart** |
| Restart entire game | Click **Restart Page** |
| Exit game | Click **Exit** |

---

## Match Control

- **Restart**  
  Restarts the current round while keeping the current score.

- **Restart Page**  
  Resets all scores and returns to the character selection screen.

- **Exit**  
  Closes the game.

---

## Game Result

At the end of each round, the game will display one of the following results:

- **Win** – The player connects five chess pieces in a row.
- **Lose** – The AI connects five chess pieces in a row.
- **Draw** – No more valid moves remain on the board.
